### 26 de fevereiro de 2025
* 🌀 Criador de Dashboards Customizáveis para Indicadores de Negociações e Leads  
Agora, os usuários podem criar dashboards personalizáveis para monitorar indicadores de negociações e leads, com opção de atualização manual dos dados através do botão “Atualizar Dados” 📈.

### 25 de fevereiro de 2025

* 🌀 Envio de eventos de conversão para o RD Marketing ao atualizar contatos via bot  
Agora, sempre que um contato for atualizado pelo bot, um evento de conversão será enviado automaticamente para o RD Marketing.
* 🌀 Integração nativa para criar/atualizar contatos no RD Marketing via bot  
Agora, é possível criar e atualizar contatos no RD Marketing automaticamente através do bot, facilitando a gestão de contatos.
